#### 移动设备分别率与rpx  如何做不同分辨率设备的自适应？
- 以ip6的物理像素750p*1334为视觉稿进行设计，而在小程序中使用rpx为单位
- ip6下 1px = 1rpx = 0.5pt
- 使用rpx, 小程序会自动在不同分辨率下进行转化，而使用px为单位不会
#### 为什么要用ipnone6的物理分别率来做设计图？
- ip6 下 1px = 1px
- ip6 plus 下 1px = 0.6rpx
